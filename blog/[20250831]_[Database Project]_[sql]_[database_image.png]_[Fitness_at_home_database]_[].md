# 운동 후 리워드를 지급하는 프로그램 데이터베이스

이번 프로젝트는 운동을 기록하면 포인트를 적립하고, 적립된 포인트로 리워드를 교환할 수 있는 서비스의 데이터베이스 설계입니다.
고객이 운동을 할 때마다 포인트가 쌓이고, 일정 포인트 이상이 되면 문화상품권, 기프티콘 등 다양한 리워드를 받을 수 있습니다.
